Conceptos Básicos de la Web

Explorando los Conceptos

¿Qué es la Web?

La Web es una red global de información accesible a través de Internet. Puedes visitar sitios web, navegar por páginas web utilizando un navegador web como Chrome, Firefox, Safari, etc.

¿Qué es una URL?

Una URL (Uniform Resource Locator) es una dirección específica que se utiliza para acceder a un sitio web en Internet, como https://www.ejemplo.com.

Frontend

El frontend se refiere a todo lo que ves y con lo que interactúas en una página web, como la estructura HTML, los estilos CSS y las interacciones con JavaScript.

Backend

El backend es la parte del servidor que gestiona las peticiones de los usuarios, procesa datos y envía respuestas a través de protocolos como HTTP.

DNS

El DNS (Domain Name System) convierte las URLs en direcciones IP para localizar los servidores web.

HTTP

HTTP (Hypertext Transfer Protocol) es el protocolo que se utiliza para transferir datos entre el navegador web y el servidor web.
